# Claude Code ガイドライン

## 基本ルール

### README の更新

ファイルの追加・削除・変更を行った場合は、必ず `README.md` を更新すること。

更新が必要なケース:
- 新しいdotfileを追加した場合 → 「構成」テーブルに追記
- ファイルを削除した場合 → 「構成」テーブルから削除
- セットアップ手順が変わる場合 → 「セットアップ」セクションを更新
- 新しい依存関係が増えた場合 → 「依存関係」セクションに追記

### セキュリティ

- トークン、パスワード、APIキーなどの機密情報は絶対にコミットしない
- `.gitconfig` などに機密情報が含まれる場合は除去してからコミット
